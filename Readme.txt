Demo for QDSFM problems with different methods

reference: Quadratic decomposable submodular function minimization, NIP2018


main.m : main file to run the demo, semisupervised learning over hypergraphs
QRCDM_cversion.cpp : RCD for QDSFM
QRCDM_AP_cversion.cpp: AP for QDSFM
PDHG_QDSFM_cversion.cpp: PDHG for QDSFM
Subgradient_QDSFM_cversion.cpp: Subgradient descent for QDSFM
clique_expansion.m: clique-expansion methods for SSL over hypergraphs (Zhou et al., NIPS07)
